1 component is the besic building block of angular, which means that an angular application is a collection of components, and one component is responsible for handling one view or part of the view.
2 A component encapsulates the data, the html markup, and the logic for a view. you can create as many components as required.
3 component is  nothing but a simple typescript class which contains the following things.
4 typescript class
5 html template/ template url
6 @component decorator with metadata.
7 every angular application has at least one component that is used to display the data on the view.
8 the component contains metadata like animation effect, style to apply the components, template, input-output, import etc.
9 These are the main building blocks of an angular application
10 when we create any component, it gets defined in NgModule.
11 angular is a component-based architecture which allows us to work on smaller and more maintainable pieces that can also be used in different places
